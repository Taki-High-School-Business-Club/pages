How To Use

What's this?

投稿の際に使用するHTMLコードを手動で入力することが非常に無駄な仕事になっていました。これを改善するためにこのフォームではいくつかの項目に入力するだけでHTMLコード自動生成することができます。

この説明の目的はこのフォームの使用方法をユーザーに伝えることです。

本文を書いてください。

活動内容

2つまで書けます

メディア

URLをフォームにいれてください

URLの形の例

YouTubeの場合

  • https://www.youtube.com/watch?v=exampleID
  • https://youtu.be/exampleID

時間とか付いてるとバグります

画像、動画,音声の場合

  • https://tacbusinesscl.com/wp-content/uploads/2021/04/example.jpg

対応ファイル形式

  • 画像
  • 動画
  • 音楽

URL発行ツール

WordPress上での編集

  1. 新規投稿の作成を選択してビジュアルエディタを開きます
  2. 「カスタムHTML」ブロックを使用し、このサイトで作ったコードをコピペしてください
  3. 投稿のタイトル、アイキャッチ画像、カテゴリ分けは手動で行ってください
  4. 後は投稿するだけです
